# Security Policy

## Supported Versions

Security fixes are provided for the latest minor release of the current major version.

| Version | Supported |
|---------|-----------|
| 2.x     | ✅        |
| 1.x     | ❌ (please upgrade — see [UPGRADING.md](UPGRADING.md)) |

## Reporting a Vulnerability

If you discover a security vulnerability, **do not open a public issue or pull request**.

Instead, report it privately using GitHub's [private vulnerability reporting](https://github.com/GrapheneICT/laravel-cognito-guard/security/advisories/new) form. We aim to acknowledge reports within 7 days.

Please include:

- A clear description of the vulnerability and its impact
- Steps to reproduce
- The affected version(s)
- Any suggested mitigation, if known

We will coordinate a fix and a coordinated disclosure timeline with you before publishing the advisory.
